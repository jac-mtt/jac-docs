# Datos de Operaciones por Aeropuertos

La serie de datos que se describen a continuación corresponden a un resumen
de las operaciones informadas en la [bitácora de los aeropuertos de Chile](./bitacora.mdx),
donde las mismas consideraciones aplican, con la diferencia que se
excluyen los sobrevuelos (W).

Los datos se encuentran en el sitio de Datos Abiertos de Gobierno
en el siguiente [enlace](https://datos.gob.cl/dataset/bitacora-aeropuertos).

Esta información reemplaza el antiguo Excel de operaciones de aeronaves:
![imagen](/img/operaciones-aeronaves-old.png)


## Actualización de Datos

Al igual que la información de bitácora, este set de datos serán actualizados
mensualmente, aproximadamente los días 10 de cada mes (`t`), con información
del periodo anterior al de la actualización (`t-1`).
