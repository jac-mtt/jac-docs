## Descarga de archivos desde la Plataforma de Datos Abiertos

El archivo de la bitácora de operaciones puede ser descargado desde el sitio web de
[Datos.Gob](https://datos.gob.cl/dataset/operaciones-aeronaves/resource/5bc8842e-d95a-4f8f-8ec6-acfbe700258a)
presionando en "Descargar":

</figure>
