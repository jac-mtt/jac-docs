# Introducción a la Documentación JAC

En este documento se describen en detalle las diversas fuentes
de datos que esta institución pone a disposición del público,
las cuales se dividen en dos categorías:
- **Series de datos**: Conjuntos de datos que se actualizan
  periódicamente, asociados a eventos o registros que ocurren
  en el tiempo.
- **Dimensiones**: Tablas que agregan atributos o contexto a las
    series de datos, permitiendo una mejor comprensión de los
    registros.

Adicionalmente, se entregan diversos ejemplos de como utilizar
estos datos mediante el uso de herramientas de análisis de datos
**(En construcción)**.

:::warning
Esta documentación, y la publicación que se hace en Datos.Gob de las tablas,
se encuentran en desarrollo, por lo que se recomienda revisar periódicamente
si existen cambios en las metodologías empleadas para generar ambas.
:::
